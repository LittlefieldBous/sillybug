Silly Bug Application - Prototype Development

The Silly Bug Application is an educational game for learners PreK4 through Third Grade. This learning application is a personal project being developed for mobile and as a web game on the website:www.bludnoodle.org

Click here to view the prototype

The Silly bug prototype was made with InVision's prototyping tools. As one clicks on the games arrows they can navigate to each page and get a glimpse into some of the features this application will have. Please feel free to contribute feedback!
